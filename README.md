# Multi-Armed-Bandits
Multi-Armed Bandits

🎰 Multi-Armed Bandit (MAB) 학습 프로젝트

이 프로젝트는 강화학습의 기초인 **Multi-Armed Bandit** 문제를 파이썬으로 구현하고, 다양한 탐색(Exploration) 전략의 성능을 비교한 과제입니다.

## 📝 프로젝트 개요
- **목표**: 5개의 슬롯머신 중 가장 보상 확률이 높은 머신을 찾아 수익을 극대화함.
- **환경**: 5개 머신의 실제 승률은 `[0.2, 0.5, 0.1, 0.8, 0.3]`이며, 에이전트는 이를 모르는 상태에서 학습을 시작함.
- **전략**: Random, Greedy, $\epsilon$-greedy (0.1, 0.01) 전략 비교.

## 📊 실험 결과 (1,000회 시도)
나의 실험에서 각 전략이 얻은 최종 누적 수익은 다음과 같습니다.
- **Random**: 387점
- **Greedy**: 781점 (초반 운이 좋아 높은 수익 기록)
- **$\epsilon$-greedy (0.1)**: 758점
- **$\epsilon$-greedy (0.01)**: 500점

## 💡 학습 질문 답변
- **Greedy 전략의 한계**: 탐험을 하지 않아 운 좋게 초반에 좋은 결과를 얻지 못하면 진짜 최고 머신을 영영 찾지 못할 위험이 큼.
- **Exploration이 필요한 이유**: 현재 알고 있는 정보가 불완전할 수 있으므로, 더 나은 대안이 있는지 지속적으로 확인해야 장기적인 수익을 보장할 수 있음.
